# 12) Полезность Sitemap

**Sitemap** — это "карта" приложения, показывающая все доступные экраны и их структуру для отладки, тестирования и документирования.

**Пример:**
```bash
npx uri-scheme open "expo-app-presentation://_sitemap" --ios
```

## 12) Полезность sitemap:

✅ **Отладка маршрутизации**
- Видно все доступные маршруты
- Проверка правильности вложенности
- Обнаружение отсутствующих routes

✅ **Проверка динамических маршрутов**
- `[id]` параметры
- `[...slug]` catch-all routes
- Optional routes `[[[...slug]]]`

✅ **Валидация групп**
- Проверка `(main)`, `(auth)`, `(tabs)` групп
- Правильность исключения из URL

✅ **Debug navigation**
- Понимание иерархии экранов
- Проверка layout структуры
- Тестирование deep links
